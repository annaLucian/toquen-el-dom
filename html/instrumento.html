Usa el teclado de la computadora para tocar batería

Explora la expresividad de nuestros platillos con sonidos vibrantes y variados. Cada tecla asignada a los platillos ofrece un rango tonal único. ¡Presiona 'Q' para el Crash 1, 'W' para el Crash 2, 'T' para el Ride, 'Z' para el Open Hi-hat y 'X' para el Close Hi-hat!

Siente el poder del ritmo con nuestros bombos. Cada golpe impulsa la energía musical. Toque el 'P' para el Kick, 'A' para el Tom 1, 'S' para el Tom 2, 'D' para el Tom 3, 'J' para el Floor Tom y 'V' para el Snare.

Usa un dispositivo moderno con un navegador actualizado. Google Chrome, Mozilla Firefox y Safari funcionan muy bien. Recuerda subir el volumen de tu dispositivo si no escuchas nada. Si crees que la batería no suena bien, prueba con unos audífonos.
